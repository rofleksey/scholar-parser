```commandline
Parse articles from scholar.google.com. Output as JSON to stdout or file.

usage: parser.py [-h] -q QUERY [-o OUTPUT] [-p PAGES]

options:
  -h, --help            show this help message and exit
  -q QUERY, --query QUERY
                        query text
  -o OUTPUT, --output OUTPUT
                        output file (or print to console if not set)
  -p PAGES, --pages PAGES
                        number of pages to query and parse
```
